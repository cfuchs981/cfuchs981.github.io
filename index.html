ArtSock

2017 • github code

I wanted to build a personal art blog and learn SQL, so I combined the two ideas and made this.

Dynamically reads in from local MySQL database on launch, formats with HTML/CSS, Pure API, and JS

SoundBoard

2017 • github code

A fun java program to experiment with audio settings. It tries to turn your keyboard into a midi fighter. Does it make you a qualified DJ? No. It doesn't. Java suffers many limitations as a language.

fiddle

2017 • github code

A thrilling 2D side scrolling in Java that only has one map thus cannot scroll to the side (YET), but is a fantastic way to spend an afternoon.

PyDaily

2017 • github code

This is a python program to experiment with Google API's and html parsing. Ultimately, it produces a daily to do list on top of collecting data from your personal Google calendar and returning a random array of recipes scraped from allrecipes.com

CSDistress Library

2017 • github code

This is a game library created for a CS study group I organized. Each member can play themselves, and it's updated each semester with games centered on the professors we have graduated from. Currently only holds one game, more in progress along with a web version.
